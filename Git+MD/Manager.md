# Описание класса Manager
    
Класс Manager является потомком класса [User](User.md) .

## Атрибуты

- +FIO:   STRING
- +ID: INT
- +PHONE: STRING
- +PASSWORD: STRING
- +SEX_MALE: BOOL
- +DATE_ADD:DATETIME

## Описание атрибутов

- +FIO:   STRING - Фамилия Имя Отчество
- +ID: INT - Индефикатор в БД
- +PHONE: STRING - Номер телефона
- +PASSWORD: STRING - Пароль
- +SEX_MALE: BOOL - Пол 
- +DATE_ADD:DATETIME - Дата добавления пользователя в бд
